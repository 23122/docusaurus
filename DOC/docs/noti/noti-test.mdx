# 노티연동 테스트

구축하신 서버URL으로 노티 테스트를 진행 할 수 있는 페이지입니다.

## 테스트정보

1. 전송되는 데이터는 `form` 데이터입니다.
2. `form` 데이터는 '노티 발송 예시' 데이터와 동일합니다.
3. 코페이에서 노티수신 판단은 `HTTP STATUS` 가 200일 경우입니다.
    1. 노티값을 받지 못했지만 `HTTP STATUS` 가 200일 경우 노티를 수신한걸로 판단합니다.
    2. 노티값을 받았지만 `HTTP STATUS` 가 200이 아닐경우 수신받지 못한걸로 판단해 재전송 됩니다.
4. 노티테스트와 실 연동에서 차이가 있을 수 있습니다.
5. 연동문의는 영업담당자에게 문의 바랍니다.
6. 기술지원 (PG개발팀 : dev@korpay.com , 070-7012-1447)
